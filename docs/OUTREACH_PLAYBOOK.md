# Outreach Playbook

This playbook turns the go-to-market plan into concrete weekly action.

## First Audience

Small security, privacy, GDPR and web agencies that already sell website checks, compliance help, client reporting or security retainers.

## Offer

```text
I can prepare a passive OSINT report for your agency domain or one client domain.

It covers public DNS, HTTPS, email posture, security headers, web exposure and an evidence graph. No exploit scanning or intrusive testing.
```

## Week 1 Message

```text
Hi {name}, I am building OSINTPRO, a passive OSINT workspace for small agencies.

It turns public domain, brand, username and wallet signals into client-ready evidence graphs and reports.

I can run one free passive report for your agency domain if you want feedback material for clients. No exploit scanning, no intrusive tests.

Live demo:
https://osintpro-48j4.onrender.com/
```

## Follow-Up

```text
Quick follow-up. The most useful part for agencies is recurring monitoring plus PDF/CSV reports that can be reused in client delivery.

If you want, send one public domain you own or manage and I will show you the type of report OSINTPRO produces.
```

## Tracking Sheet Columns

Use a spreadsheet with:

- agency name
- website
- contact
- channel
- sent date
- replied
- report requested
- demo account created
- checkout clicked
- notes

## Weekly Targets

- 25 targeted agency messages
- 1 public build update
- 1 educational post
- 1 GitHub issue update
- 1 product screenshot or sanitized example

## What To Avoid

- claiming the product hacks websites
- promising vulnerability proof without authorization
- mentioning exploit payloads
- claiming wallet identity attribution
- sending generic spam
