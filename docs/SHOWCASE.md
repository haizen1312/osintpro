# OSINTPRO Showcase

Use this page to explain OSINTPRO quickly when sharing the project.

## One-Line Pitch

OSINTPRO is a passive OSINT SaaS workspace for domain intelligence, brand monitoring, beginner-friendly web audit workflows, social username checks and blockchain wallet tracing.

## Why People Should Try It

- It runs in the browser with a live demo.
- It turns public technical signals into readable reports.
- It includes a Burp-style Web Audit Lab for beginners without exploit automation.
- It maps domains, profiles, wallets, transactions and findings into an entity graph.
- It stays within a passive OSINT safety boundary.

## Demo Workflows

### Domain Intelligence

1. Open the live demo.
2. Create a nickname account.
3. Run Domain Intel on a public domain.
4. Open the PDF report.
5. Save the domain to monitoring.

### Web Audit Lab

1. Open Web Audit Lab.
2. Click Audit OSINTPRO.
3. Read the Burp-style feature map.
4. Review the safe commands.
5. Use the glossary to understand technical terms.

### Wallet OSINT

1. Open Wallet Trace.
2. Enter a public Bitcoin or Ethereum/EVM address.
3. Review balance, recent movements and counterparties.
4. Open Entity Graph to connect wallet evidence.
