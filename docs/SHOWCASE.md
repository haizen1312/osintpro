## Launch Description

Security consultants often spend too much time turning evidence into a clean
client deliverable. OSINTPRO is built around that workflow: run a passive
investigation, review plain-English findings, connect entities in a graph and
export the result as PDF, CSV, SARIF or graph data.

Core modules:

- Domain intelligence for DNS, HTTPS, email posture, public headers and CT logs.
- Repository Audit Lab for static review, confidence filtering, SARIF export and dependency advisories.
- Social username checks across public profile surfaces.
- Wallet OSINT for public Bitcoin and Ethereum/EVM balances, movements and counterparties.
- Entity graph exports as JSON-LD, Graphviz DOT and CSV.
- Monitoring and client folders for small agency workflows.

OSINTPRO is passive-only. It does not run exploits, brute force, credential
attacks, invasive scans, unauthorized packet capture or wallet transactions.

## Why It Is Different

## Demo Workflows

### Domain Intelligence

1. Open the live demo.
2. Create a nickname account.
3. Run Domain Intel on a public domain.
4. Open the PDF report.
5. Save the domain to monitoring.

### Web Audit Lab

1. Open Web Audit Lab.
2. Click Audit OSINTPRO.
3. Read the Burp-style feature map.
4. Review the safe commands.
5. Use the glossary to understand technical terms.

### Wallet OSINT

1. Open Wallet Trace.
2. Enter a public Bitcoin or Ethereum/EVM address.
3. Review balance, recent movements and counterparties.
4. Open Entity Graph to connect wallet evidence.
